DSL BASED FRAMEWORK

Galite provide a DSL layer for developer to use. It make generating application from model very easy. Using few lines of code, Galite generate a nice user interface with backend fully managed. 


  class ClientForm : Form() {
    override val locale = Locale.UK
    override val title = "form-title"
    val action = menu("Action")

    val helpForm = actor(
        ident = "helpForm",
        menu = action,
        label = "Help",
        help = " Help"
    ) {
        key = Key.F1   // key is optional here
        icon = "help"  // icon is optional here
    }

    val helpCmd = command(item = helpForm) {
        action = {
        showHelp()
        }
    }

    val clientsPage= page("Clients")
    val block = clientsPage.insertBlock(Clients())
  }
