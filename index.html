Opensource framework that allows users to build business applications made of forms, reports and charts using a simple and creative syntax based on a domain specific language.

Get started   Github   Maven

Generic

Make it easy to build a full fledged application based on business requirement without the need of creating tailor components.

Extendible

It provide a collections of component that can be combined in order to build application in a timely manner.

Developer friendly

Galite provide a domain specific language that is easy to use in combination with the Koltin programming language.

DSL BASED FRAMEWORK

Galite provides a DSL layer for developer to use. It make generating application from model very easy. Using few lines of code, Galite generate a nice user interface with backend fully managed. 

        
class Clients : Form() {
  override val title = "Clients form"

  val block = insertBlock(ClientBlock())

  inner class ClientBlock : Block(1, 1, "Clients") {

    val id = hidden(domain = INT(20)) {
      label = "ID"
      help = "The user id"
      columns(Clients.id) {
        index = unique
      }
    }

    val name = visit(domain = STRING(20), position = at(1, 1)) {
      label = "name"
      help = "name"
      columns(Clients.name)
    }

    val createdOn = visit(domain = TIMESTAMP, position = at(2, 1)) {
      label = "createdOn"
      help = "createdOn"
      columns(Clients.createdOn)
    }
  }
}
